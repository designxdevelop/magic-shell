---
title: Keyboard Shortcuts
description: Complete reference for Magic Shell TUI keyboard shortcuts.
---

Magic Shell follows OpenCode-style TUI shortcuts where they map cleanly to this app.

## Core Shortcuts

| Shortcut | Action |
| --- | --- |
| `Ctrl+P` | Open Commands |
| `Ctrl+X M` | Change model |
| `Ctrl+X T` | Change theme |
| `Ctrl+X Q` | Exit |
| `Ctrl+T` | Cycle thinking level |
| `Ctrl+C` | Exit, or cancel/close the active dialog first |
| `Ctrl+D` | Exit |
| `Esc` | Close dialogs or cancel pending commands |

## Commands

Press `Ctrl+P` to open Commands. Type to filter the list, use arrow keys to move, and press Enter to run the selected action.

## Slash Commands

Type these in the input field as alternatives to shortcuts:

| Command | Action |
| --- | --- |
| `!help` or `/help` | Show help |
| `!model` or `/models` | Change model |
| `!provider` or `/providers` | Switch provider |
| `!theme` or `/themes` | Change theme |
| `!dry` or `/dry` | Toggle dry-run |
| `!thinking` or `/thinking` | Cycle thinking level |
| `!config` or `/config` | Show config |
| `!history` or `/history` | Show history |
| `!clear` or `/clear` | Clear output |
| `!exit` or `/exit` | Exit |

## Input Shortcuts

| Shortcut | Action |
| --- | --- |
| `Enter` | Submit |
| `Shift+Enter` | Insert newline |
| `Ctrl+Enter` | Insert newline |
| `Alt+Enter` | Insert newline |
| `Ctrl+J` | Insert newline |
| `Ctrl+A` | Move to start of line |
| `Ctrl+E` | Move to end of line |
| `Ctrl+U` | Clear line |
| `Ctrl+W` | Delete word before cursor |
| `Ctrl+K` | Delete to end of line |
