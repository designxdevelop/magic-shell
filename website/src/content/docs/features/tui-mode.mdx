Magic Shell's TUI mode provides a rich interactive experience for translating and executing commands.

```bash
mshell
```

## Interface Overview

- **Header**: Current model, provider, and mode indicators
- **Output area**: Command translations, safety analysis, and execution results
- **Input area**: Multi-line prompt for natural-language requests or slash commands
- **Status bar**: Current provider, model, safety level, thinking level, and mode

## Keyboard Shortcuts

Magic Shell follows OpenCode-style TUI shortcuts where they make sense for this app:

| Shortcut | Action |
| --- | --- |
| `Ctrl+P` | Open Commands |
| `Ctrl+X M` | Change model |
| `Ctrl+X T` | Change theme |
| `Ctrl+X Q` | Exit TUI |
| `Ctrl+T` | Cycle thinking level |
| `Ctrl+C` | Exit, or cancel/close the active dialog first |
| `Ctrl+D` | Exit |
| `Esc` | Close dialogs or cancel pending commands |

## Slash Commands

Use `!` or `/` commands directly in the input:

| Command | Action |
| --- | --- |
| `!help` or `/help` | Show help |
| `!model` or `/models` | Change model |
| `!provider` or `/providers` | Switch provider |
| `!theme` or `/themes` | Change theme |
| `!dry` or `/dry` | Toggle dry-run |
| `!config` or `/config` | Show config |
| `!history` or `/history` | Show history |
| `!clear` or `/clear` | Clear output |

## Dry-Run Mode

Dry-run mode translates and analyzes commands without executing them.

```bash
!dry
```

## History

Press `Ctrl+P` and choose **Show History**, or type `!history`, to review prior commands and outputs.

## Model Switching

<Steps>
1. Press `Ctrl+X M` or type `/models`
2. Use arrow keys to select a model
3. Press Enter to confirm
</Steps>

## Provider Switching

Press `Ctrl+P` and choose **Switch Provider**, or type `/providers`.

## Tips

1. Use dry-run for complex or destructive commands.
2. Check history for similar past commands.
3. Switch models if translations are not accurate.
4. Use Commands with `Ctrl+P` to discover available actions.

## Exiting TUI

- `Ctrl+C` exits when no dialog or confirmation is active
- `Ctrl+D` exits
- `Ctrl+X Q` exits
