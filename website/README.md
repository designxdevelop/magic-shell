# Magic Shell Website

Marketing site and Starlight documentation for Magic Shell.

## v1.0 Documentation Prep

The site now describes the Magic Shell 1.0 terminal UX:

- Unified modal palettes for commands, slash commands, models, providers, themes, safety, thinking, onboarding, and API-key setup.
- Slash commands such as `/models`, `/providers`, `/themes`, `/safety`, and `/thinking` open the same actions exposed in `Ctrl+P`.
- Updated keyboard shortcut docs for `Ctrl+P`, `Ctrl+X M`, `Ctrl+X T`, `Ctrl+Y`, `Ctrl+T`, `Esc`, `Ctrl+C`, `Ctrl+D`, and `Ctrl+X Q`.
- Changelog entry for `v1.0.0`.

## Commands

```bash
bun install
bun run dev
bun run build
bun run preview
```

Editable docs live in `website/src/content/docs/`. Landing page copy lives in `website/src/pages/index.astro` and `website/src/components/`.
