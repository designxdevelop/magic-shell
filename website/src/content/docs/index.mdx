## Quick Start

Magic Shell 1.0 ships a more cohesive terminal experience: unified command palettes, slash commands that open the same actions as shortcuts, explicit safety and thinking pickers, clearer model/provider descriptions, and consistent modal behavior across setup and daily use.

```bash
# Install Magic Shell
bun add -g @austinthesing/magic-shell@latest

# Configure your API key
msh --setup

# Launch the v1.0 interactive TUI
mshell

# Or use quick CLI mode
msh "find all files larger than 100MB"
```
